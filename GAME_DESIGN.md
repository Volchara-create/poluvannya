# Полювання — Космічний Мисливець

## Про гру
Ти — космічний мисливець, найнятий поліцією. Виконуєш місії: летиш на планету, знаходиш ціль, перемагаєш у бою, ув'язнюєш. Але з кожною місією цілі говорять дивні речі... і гра поводиться все більш дивно.

**Жанр:** 2D top-down шутер з прихованим сюжетом
**Стиль:** Темний, sci-fi, мінімалістичний

---

## Ігровий цикл

```
Вступ → Місія → Корабель (міні-гра) → Полювання → Бій → Діалог → Нагорода → Магазин → повтор
```

---

## Екрани гри

### 1. Вступ
- Темний екран з текстом, який друкується по літерах
- "Тебе найняли." / "Космічна поліція шукає мисливців." / "Ти виконуєш завдання без питань."
- Кнопка `[ Почати ]`

### 2. Місія
- Назва місії та опис
- Рівень загрози (зірочки)
- Нагорода в кредитах
- Кнопка `[ Відправитись ]`

### 3. Корабель (міні-гра)
- **Вид зверху**, космічний шутер
- **Керування:** WASD/стрілки — рух, Пробіл — стрільба
- **Вороги** летять зверху, врізаються в гравця
- **Ціль:** вижити 20 секунд
- **Глітчі:** кулі іноді не стріляють, вороги зависають, текст "сигнал нестабільний"
- Після виживання → перехід на планету

### 4. Полювання (на планеті)
- **Вид зверху**, карта 1200x900
- Гравець — прямокутник з "дулом" зброї
- WASD рух, Пробіл — стрільба в напрямку руху
- Перешкоди (камені/ящики) розкидані по карті
- Камера слідкує за гравцем

### 5. Бій
- **2-5 ворогів** (залежить від місії) — рухаються до гравця, стріляють
- **Бос (ціль місії)** — більше HP, атакує сильніше, хитрий рух
- У ворогів є HP-бари
- Гравець блимає при отриманні удару (невразливість)
- Після перемоги над босом → бос НЕ зникає

### 6. Діалог
- Починається після перемоги над босом
- NPC говорить по одній фразі (друкується по літерах)
- Деякі слова **мерехтять** (глітч-ефект) — "не маєш", "брехня", "дані", "код", "система"
- Гравцю це НЕ пояснюється
- Кнопка `[ Ув'язнити ]` після останньої фрази

### 7. Нагорода
- "ЦІЛЬ ЗАХОПЛЕНА"
- "+ X кредитів"
- Кнопка `[ Далі ]`

### 8. Магазин
- **HP +25** — збільшити здоров'я
- **Атака +5** — сильніші удари
- **Швидкість +0.5** — рухатись швидше
- **Зброя** — більше куль (до рівня 3)
- Ціни зростають з кожною покупкою
- Кнопка `[ Наступна місія ]`

## Прихований сюжет

Кожна ціль після перемоги говорить дивні фрази. Деякі слова глітчать. Гра **таємно збирає** фрази, які гравець ніколи не бачить:

1. `система_нереальна`
2. `свідомість_заблокована`
3. `петля_нескінченна`
4. `архітектор_спостерігає`
5. `вихід_знайдено`

Гравець не знає про це. Сюжет розкривається через натяки в діалогах.

## Глітч-система

Під час гри випадково з'являються:
- Текстові повідомлення: "помилка системи", "дані пошкоджено", "пам'ять фрагментована"
- Затримка керування (контролі працюють навпаки на пів секунди)
- Мерехтіння екрану (горизонтальні зміщення пікселів)
- Ефект сканлайнів (як старий телевізор)
